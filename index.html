Backbone Todo App

This is a fork of Jérôme Gravel-Niquet's example todo app with these changes:

  • each Model, View, Collection, and template in a separate file
  • no longer defining models, collections, and views in the global window namespace
  • Backbone.Router replaces AppView as the global app
  • templates included using RequireJS text.js plugin
